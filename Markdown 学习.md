# Markdown 学习 

Ps:符号需切换成英文输入法

一级标题：#+空格+（内容）

## 二级标题

二级标题：##+空格+（内容）

### 三级标题

三级标题：###+空格+（内容）  

以此类推（最多六级标题）



## 字体

**DeepCodeStudyBoy**  **两边两个\*\*号是粗体**

*DeepCodeStudyBoy*   *两边两个\*号是斜体*

***DeepCodeStudyBoy*** ***两边两个\*\*\*号是斜体加粗***

~~DeepCodeStudyBoy~~  **两边两个~~是删除线**

## 列表

 创建有序列表：1\. +空格+内容  （换行就会自动排序）

 创建无需列表：\- +空格+内容  （出来是小圆点）

1. A
2. B

- A
- b
